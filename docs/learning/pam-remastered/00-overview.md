# PAM Remastered — Overview

**Effect:** Part 3 of the Second Reality demo (1993, Future Crew)
**What you see:** A volcanic explosion erupts over the ALKU landscape — a molten lava core with orbiting embers, surrounded by raymarched volumetric smoke that expands outward, framed by a purple horizon glow and twinkling stars
**Duration:** 4 seconds (78.0 s – 82.0 s in the full demo timeline)

---

## What This Guide Covers

This guide unpacks the remastered PAM effect layer by layer. By the end you
will understand:

- How a 41-frame FLI explosion (by TRUG / WILDFIRE) is stored as Base64 and decoded via a custom RLE codec
- How a pre-authored **PALETTE_FADE** curve creates dramatic white-flash transitions at the start and end
- How 320×200 indexed frames are converted to RGBA and uploaded as GPU textures each frame
- How procedural lava, volumetric smoke, and atmosphere are composited in a single fragment shader
- How a dual-tier bloom pipeline adds cinematic glow with beat reactivity

Each layer builds on the last. You can stop at any point and still have learned
something valuable.

## File Map

The effect lives in three source files plus shared WebGL helpers:

```
src/effects/pam/
  data.js                 — Base64 animation data + 256-colour palette
                            (auto-generated by tools/extract-pam-data.mjs)

  effect.js               — Classic variant (185 lines)
                            CPU software renderer, 320×200, palette fading.
                            Faithful 1:1 recreation of the 1993 original.

  effect.remastered.js    — Remastered variant (993 lines)
                            Procedural lava core, volumetric blast,
                            atmosphere overlays, dual-tier bloom.

src/core/
  webgl.js                — Shared WebGL2 helpers
                            createProgram(), createFullscreenQuad(), etc.
```

## Architecture

Both variants decode frames from the same `ANI_B64` data in `data.js`. The
classic plays all 41 frames as a video. The remastered uses only frames 0–6
as a subtle background reference, then paints procedural lava, volumetric
smoke, and atmospheric layers on top via GPU shaders.

```
                 data.js
         (ANI_B64, PALETTE)
              /           \
             v             v
      effect.js      effect.remastered.js
   (41-frame video)  (7 frames as backdrop +
    palette fading    procedural lava core +
    CPU → texture)    volumetric raymarched blast +
                      horizon glow + stars +
                      dual-tier bloom)
```

The key insight: the classic is a **video player** — it decodes pre-rendered
frames and applies palette fading. The remastered is a **real-time shader
effect** — it uses the original frames only as a faint background reference,
then generates every visual element procedurally at native resolution.

## The Effect Interface

Every effect in this project exports three methods:

```javascript
export default {
  init(gl) { },             // called once — set up shaders, buffers, textures
  render(gl, t, beat, params) { },  // called every frame
  destroy(gl) { },          // called when the effect is unloaded
}
```

Where:
- `gl` — a WebGL2 rendering context
- `t` — seconds elapsed since this clip started (not global time)
- `beat` — 0.0 to 1.0 position within the current musical bar
- `params` — key-value object of tunable parameters from the editor

The remastered variant exports a `params` array describing 28 tunable knobs
(explosion speed, core intensity, smoke detail, ember orbits, bloom, etc.)
that the editor auto-generates UI controls for.
